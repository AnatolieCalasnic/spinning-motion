# Spinning Motion

Spinning Motion is a full-stack web application designed to bring something new to the vinyl record marketplace, combining modern web technologies with a passion for music.

### Available scripts:

`npm start: Launch development server`

`npm run build: Create production build`

`npm test: Run Cypress E2E tests`
